\# Projeto Página de Login



Este projeto foi desenvolvido com foco em prática de desenvolvimento web utilizando HTML5, CSS3 e JavaScript. A aplicação consiste em uma interface moderna de página de login, com design responsivo e organização visual voltada para uma boa experiência do usuário.



\## Tecnologias utilizadas



\* HTML5

\* CSS3

\* JavaScript



\## Funcionalidades



\* Estrutura semântica em HTML

\* Estilização moderna com CSS

\* Layout responsivo

\* Campos de login e senha

\* Botão de acesso

\* Manipulação do DOM com JavaScript

\* Organização visual para estudos de front-end



\## Objetivo do projeto



O objetivo deste projeto é praticar conceitos fundamentais do desenvolvimento web, incluindo estruturação de páginas, estilização, responsividade, manipulação do DOM e organização de arquivos para publicação no GitHub.
